## Instalación

1. Clonar el repositorio:
    ```bash
    git clone https://github.com/inigomurga/persistencia-G2.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd persistencia-G2
    ```
3. Construir y lanza los contenedores Docker:
    ```bash
    docker-compose up -d
    ```
